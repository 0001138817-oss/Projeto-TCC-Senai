# 📊 Projeto-TCC-Senai
**Gestão Inteligente do Ciclo de Vida de Produtos**

## 📌 Sobre
Repositório do TCC do SENAI baseado em demandas reais da plataforma Saga Senai.

## 🎯 Objetivo
Criar um sistema para gerenciar o ciclo de vida de produtos com foco em organização e tomada de decisão baseada em dados.

## 🏢 Desafio
A empresa enfrenta dificuldades para organizar e controlar o ciclo de vida de seus produtos, utilizando processos manuais que comprometem a agilidade e a tomada de decisão baseada em dados.

**Benefícios esperados:**
- Reduzir tempo de lançamento
- Melhorar organização
- Apoiar decisões com dados
- Centralizar informações

**Problemas atuais:**
- Processos manuais
- Falta de versionamento
- Sem dashboards
- Perda de informações

**Restrições:**
- Interface simples
- Exportação PDF/Excel
- Baixo custo
- Segurança dos dados

## 🛠️ Tecnologias
- Frontend: HTML, CSS, JavaScript
- Backend: Java
- Banco de Dados: MySQL
- Versionamento: GitHub

## ⚙️ Funcionalidades
- Cadastro de produtos
- Controle de ciclo de vida
- Versionamento
- Dashboard
- Relatórios
- Controle de acesso
